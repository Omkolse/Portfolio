# ROS-Scout: 3D Printable Files

This folder contains all the `.stl` files required to 3D print your own ROS-Scout robot structure. 

## Overview
These files include the main chassis, motor brackets, and mounts for the sensors (Arducam ToF, MPU9250 IMU) and the Raspberry Pi 4. They are designed to be easily printable on most standard FDM 3D printers.

## Printing Guidelines
- **Material:** PLA, PETG, or ABS (PETG is recommended for a good balance of strength and durability).
- **Infill:** 15-25% should be sufficient for structural integrity.
- **Supports:** Check your slicer preview; some overhanging mounts may require standard supports.

Once printed, you can assemble the robot using standard M2/M2.5/M3 hardware to secure the electronics and motors.

If you need to make modifications to the design for custom hardware, please refer to the [SolidWorks CAD Folder](../ROS_Scout_Cad) for the original editable models.
